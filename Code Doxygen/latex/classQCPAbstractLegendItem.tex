\doxysubsection{Detailed Description}
The abstract base class for all entries in a \doxylink{classQCPLegend}{QCPLegend}. 

It defines a very basic interface for entries in a \doxylink{classQCPLegend}{QCPLegend}. For representing plottables in the legend, the subclass \doxylink{classQCPPlottableLegendItem}{QCPPlottable\+Legend\+Item} is more suitable.

Only derive directly from this class when you need absolute freedom (e.\+g. a custom legend entry that\textquotesingle{}s not even associated with a plottable).

\item draw (from \doxylink{classQCPLayerable}{QCPLayerable})\end{DoxyItemize}
