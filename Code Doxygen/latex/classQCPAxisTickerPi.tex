\doxysubsection{Detailed Description}
Specialized axis ticker to display ticks in units of an arbitrary constant, for example pi. 



This \doxylink{classQCPAxisTicker}{QCPAxis\+Ticker} subclass generates ticks that are expressed with respect to a given symbolic constant with a numerical value specified with \doxylink{classQCPAxisTickerPi_a36ce0651d2ec92edd36feac1619c2468}{set\+Pi\+Value} and an appearance in the tick labels specified with \doxylink{classQCPAxisTickerPi_acfdcd4758a393bde4be12a50fb2017b5}{set\+Pi\+Symbol}.

Ticks may be generated at fractions of the symbolic constant. How these fractions appear in the tick label can be configured with \doxylink{classQCPAxisTickerPi_a760c8af6ca68178e607556c4e5049d71}{set\+Fraction\+Style}.
