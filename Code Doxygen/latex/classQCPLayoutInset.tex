\doxysubsection{Detailed Description}
A layout that places child elements aligned to the border or arbitrarily positioned. 

Elements are placed either aligned to the border or at arbitrary position in the area of the layout. Which placement applies is controlled with the \doxylink{classQCPLayoutInset_a8b9e17d9a2768293d2a7d72f5e298192}{Inset\+Placement} (\doxylink{classQCPLayoutInset_a63298830744d5d8c5345511c00fd2144}{set\+Inset\+Placement}).

Elements are added via \doxylink{classQCPLayoutInset_ad61529eb576af7f04dff94abb10c745a}{add\+Element(\+QCPLayout\+Element \texorpdfstring{$\ast$}{*}element, Qt\+::\+Alignment alignment)} or \doxylink{classQCPLayoutInset_a8ff61fbee4a1f0ff45c398009d9f1e56}{add\+Element(\+QCPLayout\+Element \texorpdfstring{$\ast$}{*}element, const QRect\+F \&rect)}. If the first method is used, the inset placement will default to \doxylink{classQCPLayoutInset_a8b9e17d9a2768293d2a7d72f5e298192aa81e7df4a785ddee2229a8f47c46e817}{ip\+Border\+Aligned} and the element will be aligned according to the {\itshape alignment} parameter. The second method defaults to \doxylink{classQCPLayoutInset_a8b9e17d9a2768293d2a7d72f5e298192aa4802986ea2cea457f932b115acba59e}{ip\+Free} and allows placing elements at arbitrary position and size, defined by {\itshape rect}.

The alignment or rect can be set via \doxylink{classQCPLayoutInset_a62882a4f9ad58bb0f53da12fde022abe}{set\+Inset\+Alignment} or \doxylink{classQCPLayoutInset_aa487c8378a6f9533567a2e6430099dc3}{set\+Inset\+Rect}, respectively.

This is the layout that every \doxylink{classQCPAxisRect}{QCPAxis\+Rect} has as \doxylink{classQCPAxisRect_a949f803466619924c7018df4b511ae10}{QCPAxis\+Rect\+::inset\+Layout}.
