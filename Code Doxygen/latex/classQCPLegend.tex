\doxysubsection{Detailed Description}
Manages a legend inside a \doxylink{classQCustomPlot}{QCustom\+Plot}. 

A legend is a small box somewhere in the plot which lists plottables with their name and icon.

A legend is populated with legend items by calling \doxylink{classQCPAbstractPlottable_aa64e93cb5b606d8110d2cc0a349bb30f}{QCPAbstract\+Plottable\+::add\+To\+Legend} on the plottable, for which a legend item shall be created. In the case of the main legend (\doxylink{classQCustomPlot_a4eadcd237dc6a09938b68b16877fa6af}{QCustom\+Plot\+::legend}), simply adding plottables to the plot while \doxylink{classQCustomPlot_ad8858410c2db47b7104040a3aa61c3fc}{QCustom\+Plot\+::set\+Auto\+Add\+Plottable\+To\+Legend} is set to true (the default) creates corresponding legend items. The legend item associated with a certain plottable can be removed with \doxylink{classQCPAbstractPlottable_a3cc235007e2343a65ad4f463767e0e20}{QCPAbstract\+Plottable\+::remove\+From\+Legend}. However, \doxylink{classQCPLegend}{QCPLegend} also offers an interface to add and manipulate legend items directly\+: \doxylink{classQCPLegend_acfe9694c45104a3359d3806ed366fcf7}{item}, \doxylink{classQCPLegend_a91e790002d8bf15a20628a8e8841e397}{item\+With\+Plottable}, \doxylink{classQCPLegend_a57ab86ab8b2a3762d4c1455eb5452c88}{item\+Count}, \doxylink{classQCPLegend_a3ab274de52d2951faea45a6d975e6b3f}{add\+Item}, \doxylink{classQCPLegend_ac91595c3eaa746fe6321d2eb952c63bb}{remove\+Item}, etc.

Since \doxylink{classQCPLegend}{QCPLegend} derives from \doxylink{classQCPLayoutGrid}{QCPLayout\+Grid}, it can be placed in any position a \doxylink{classQCPLayoutElement}{QCPLayout\+Element} may be positioned. The legend items are themselves \doxylink{classQCPLayoutElement}{QCPLayout\+Elements} which are placed in the grid layout of the legend. \doxylink{classQCPLegend}{QCPLegend} only adds an interface specialized for handling child elements of type \doxylink{classQCPAbstractLegendItem}{QCPAbstract\+Legend\+Item}, as mentioned above. In principle, any other layout elements may also be added to a legend via the normal \doxylink{classQCPLayoutGrid}{QCPLayout\+Grid} interface. See the special page about \doxylink{}{The Layout System} for examples on how to add other elements to the legend and move it outside the axis rect.

Use the methods \doxylink{classQCPLayoutGrid_affc2f3cfd22f28698c5b29b960d2a391}{set\+Fill\+Order} and \doxylink{classQCPLayoutGrid_ab36af18d77e4428386d02970382ee598}{set\+Wrap} inherited from \doxylink{classQCPLayoutGrid}{QCPLayout\+Grid} to control in which order (column first or row first) the legend is filled up when calling \doxylink{classQCPLegend_a3ab274de52d2951faea45a6d975e6b3f}{add\+Item}, and at which column or row wrapping occurs. The default fill order for legends is \doxylink{classQCPLayoutGrid_a7d49ee08773de6b2fd246edfed353ccaa0202730954e26c474cc820164aedce3e}{fo\+Rows\+First}.

By default, every \doxylink{classQCustomPlot}{QCustom\+Plot} has one legend (\doxylink{classQCustomPlot_a4eadcd237dc6a09938b68b16877fa6af}{QCustom\+Plot\+::legend}) which is placed in the inset layout of the main axis rect (\doxylink{classQCPAxisRect_a949f803466619924c7018df4b511ae10}{QCPAxis\+Rect\+::inset\+Layout}). To move the legend to another position inside the axis rect, use the methods of the \doxylink{classQCPLayoutInset}{QCPLayout\+Inset}. To move the legend outside of the axis rect, place it anywhere else with the QCPLayout/ \doxylink{classQCPLayoutElement}{QCPLayout\+Element} interface.

{\footnotesize\ttfamily void QCPLegend\+::set\+Selected\+Parts (\begin{DoxyParamCaption}\item[{const Selectable\+Parts \&}]{selected}{}\end{DoxyParamCaption})}

Sets the selected state of the respective legend parts described by \doxylink{classQCPLegend_a5404de8bc1e4a994ca4ae69e2c7072f1}{Selectable\+Part}. When a part is selected, it uses a different pen/font and brush. If some legend items are selected and {\itshape selected} doesn\textquotesingle{}t contain \doxylink{classQCPLegend_a5404de8bc1e4a994ca4ae69e2c7072f1a768bfb95f323db4c66473375032c0af7}{sp\+Items}, those items become deselected.

The entire selection mechanism is handled automatically when \doxylink{classQCustomPlot_a5ee1e2f6ae27419deca53e75907c27e5}{QCustom\+Plot\+::set\+Interactions} contains i\+Select\+Legend. You only need to call this function when you wish to change the selection state manually.

This function can change the selection state of a part even when \doxylink{classQCPLegend_a9ce60aa8bbd89f62ae4fa83ac6c60110}{set\+Selectable\+Parts} was set to a value that actually excludes the part.

emits the \doxylink{classQCPLegend_a82c88464edac07a9eefaf3906268df3b}{selection\+Changed} signal when {\itshape selected} is different from the previous selection state.

Note that it doesn\textquotesingle{}t make sense to set the selected state \doxylink{classQCPLegend_a5404de8bc1e4a994ca4ae69e2c7072f1a768bfb95f323db4c66473375032c0af7}{sp\+Items} here when it wasn\textquotesingle{}t set before, because there\textquotesingle{}s no way to specify which exact items to newly select. Do this by calling \doxylink{classQCPAbstractLegendItem_a6eed93b0ab99cb3eabb043fb08179c2b}{QCPAbstract\+Legend\+Item\+::set\+Selected} directly on the legend item you wish to select.
