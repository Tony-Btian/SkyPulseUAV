\doxysubsection{Detailed Description}
Represents the visual appearance of scatter points. 

This class holds information about shape, color and size of scatter points. In plottables like \doxylink{classQCPGraph}{QCPGraph} it is used to store how scatter points shall be drawn. For example, \doxylink{classQCPGraph_a12bd17a8ba21983163ec5d8f42a9fea5}{QCPGraph\+::set\+Scatter\+Style} takes a \doxylink{classQCPScatterStyle}{QCPScatter\+Style} instance.

A scatter style consists of a shape (\doxylink{classQCPScatterStyle_a7c641c4d4c6d29cb705d3887cfce91c1}{set\+Shape}), a line color (\doxylink{classQCPScatterStyle_a761f1f229cc0ca4703e1e2b89f6dd1ba}{set\+Pen}) and possibly a fill (\doxylink{classQCPScatterStyle_a74d692aaeb8d4b36d6f7d510e44264b1}{set\+Brush}), if the shape provides a fillable area. Further, the size of the shape can be controlled with \doxylink{classQCPScatterStyle_aaefdd031052892c4136129db68596e0f}{set\+Size}.\hypertarget{classQCPScatterStyle_QCPScatterStyle-defining}{}\doxysubsection{\texorpdfstring{Specifying a scatter style}{Specifying a scatter style}}\label{classQCPScatterStyle_QCPScatterStyle-defining}
You can set all these configurations either by calling the respective functions on an instance\+:

\end{DoxyCodeInclude}


Or you can use one of the various constructors that take different parameter combinations, making it easy to specify a scatter style in a single call, like so\+: 
\begin{DoxyCodeInclude}{0}

\end{DoxyCodeInclude}
\hypertarget{classQCPScatterStyle_QCPScatterStyle-undefinedpen}{}\doxysubsection{\texorpdfstring{Leaving the color/pen up to the plottable}{Leaving the color/pen up to the plottable}}\label{classQCPScatterStyle_QCPScatterStyle-undefinedpen}
There are two constructors which leave the pen undefined\+: \doxylink{classQCPScatterStyle_a8836018d9ad83ccd8870de8315c1be73}{QCPScatter\+Style()} and \doxylink{classQCPScatterStyle_a003d92f74f4561eda111862eadd62f28}{QCPScatter\+Style(\+Scatter\+Shape shape, double size)}. If those constructors are used, a call to \doxylink{classQCPScatterStyle_a47077eb6450fe9a788f833e4ec1b1d5a}{is\+Pen\+Defined} will return false. It leads to scatter points that inherit the pen from the plottable that uses the scatter style. Thus, if such a scatter style is passed to \doxylink{classQCPGraph}{QCPGraph}, the line color of the graph (\doxylink{classQCPAbstractPlottable_ab74b09ae4c0e7e13142fe4b5bf46cac7}{QCPGraph\+::set\+Pen}) will be used by the scatter points. This makes it very convenient to set up typical scatter settings\+:

\end{DoxyCodeInclude}


Notice that it wasn\textquotesingle{}t even necessary to explicitly call a \doxylink{classQCPScatterStyle}{QCPScatter\+Style} constructor. This works because \doxylink{classQCPScatterStyle}{QCPScatter\+Style} provides a constructor that can transform a \doxylink{classQCPScatterStyle_adb31525af6b680e6f1b7472e43859349}{Scatter\+Shape} directly into a \doxylink{classQCPScatterStyle}{QCPScatter\+Style} instance (that\textquotesingle{}s the \doxylink{classQCPScatterStyle_a003d92f74f4561eda111862eadd62f28}{QCPScatter\+Style(\+Scatter\+Shape shape, double size)} constructor with a default for {\itshape size}). In those cases, C++ allows directly supplying a \doxylink{classQCPScatterStyle_adb31525af6b680e6f1b7472e43859349}{Scatter\+Shape}, where actually a \doxylink{classQCPScatterStyle}{QCPScatter\+Style} is expected.\hypertarget{classQCPScatterStyle_QCPScatterStyle-custompath-and-pixmap}{}\doxysubsection{\texorpdfstring{Custom shapes and pixmaps}{Custom shapes and pixmaps}}\label{classQCPScatterStyle_QCPScatterStyle-custompath-and-pixmap}
\doxylink{classQCPScatterStyle}{QCPScatter\+Style} supports drawing custom shapes and arbitrary pixmaps as scatter points.

For custom shapes, you can provide a QPainter\+Path with the desired shape to the \doxylink{classQCPScatterStyle_a96a3e949f90b2afe5677ca9412a12a1e}{set\+Custom\+Path} function or call the constructor that takes a painter path. The scatter shape will automatically be set to \doxylink{classQCPScatterStyle_adb31525af6b680e6f1b7472e43859349a15d9bcfd9de94edda949006529f9219d}{ss\+Custom}.

For pixmaps, you call \doxylink{classQCPScatterStyle_a5fb611d46acfac520d7b89a1c71d9246}{set\+Pixmap} with the desired QPixmap. Alternatively you can use the constructor that takes a QPixmap. The scatter shape will automatically be set to \doxylink{classQCPScatterStyle_adb31525af6b680e6f1b7472e43859349a8718b849ca7c307b07b8e091efb0c31e}{ss\+Pixmap}. Note that \doxylink{classQCPScatterStyle_aaefdd031052892c4136129db68596e0f}{set\+Size} does not influence the appearance of the pixmap.

Defines the shape used for scatter points.
