\doxysubsection{Detailed Description}
Provides rect/rubber-\/band data selection and range zoom interaction. 

\doxylink{classQCPSelectionRect}{QCPSelection\+Rect} is used by \doxylink{classQCustomPlot}{QCustom\+Plot} when the \doxylink{classQCustomPlot_a810ef958ebe84db661c7288b526c0deb}{QCustom\+Plot\+::set\+Selection\+Rect\+Mode} is not \doxylink{namespaceQCP_ac9aa4d6d81ac76b094f9af9ad2d3aacfa9032f170490d67240a6c68c2638ffab1}{QCP\+::srm\+None}. When the user drags the mouse across the plot, the current selection rect instance (\doxylink{classQCustomPlot_a0c09f96df15faa4799ad7051bb16cf33}{QCustom\+Plot\+::set\+Selection\+Rect}) is forwarded these events and makes sure an according rect shape is drawn. At the begin, during, and after completion of the interaction, it emits the corresponding signals \doxylink{classQCPSelectionRect_a7b7162d19f4f2174d3644ff1a5d335aa}{started}, \doxylink{classQCPSelectionRect_a1bab11026bca52740c2e6682623e6964}{changed}, \doxylink{classQCPSelectionRect_aeb82009393c90130102dccf36477b906}{canceled}, and \doxylink{classQCPSelectionRect_a15a43542e1f7b953a44c260b419e6d2c}{accepted}.

The \doxylink{classQCustomPlot}{QCustom\+Plot} instance connects own slots to the current selection rect instance, in order to react to an accepted selection rect interaction accordingly.

\doxylink{classQCPSelectionRect_ad27c1569c6ea8fa48e24b81e2a302df3}{is\+Active} can be used to check whether the selection rect is currently active. An ongoing selection interaction can be cancelled programmatically via calling \doxylink{classQCPSelectionRect_af67bc58f4f5ce9a4dc420b9c42de235a}{cancel} at any time.

The appearance of the selection rect can be controlled via \doxylink{classQCPSelectionRect_ada20b7fb1b2dcbe50523262636b06963}{set\+Pen} and \doxylink{classQCPSelectionRect_ab0c66f1484418782efa01f4153611080}{set\+Brush}.

If you wish to provide custom behaviour, e.\+g. a different visual representation of the selection rect (QCPSelection\+Rect\+::draw), you can subclass \doxylink{classQCPSelectionRect}{QCPSelection\+Rect} and pass an instance of your subclass to \doxylink{classQCustomPlot_a0c09f96df15faa4799ad7051bb16cf33}{QCustom\+Plot\+::set\+Selection\+Rect}.
