\doxysubsection{Detailed Description}
Describes a data range given by begin and end index. 

\doxylink{classQCPDataRange}{QCPData\+Range} holds two integers describing the begin (\doxylink{classQCPDataRange_a54ff59048e01e46ac4aefafc844626c6}{set\+Begin}) and end (\doxylink{classQCPDataRange_a277f1a9eafe70b9184d9c00b641ae5de}{set\+End}) index of a contiguous set of data points. The {\itshape end} index corresponds to the data point just after the last data point of the data range, like in standard iterators.

Data Ranges are not bound to a certain plottable, thus they can be freely exchanged, created and modified. If a non-\/contiguous data set shall be described, the class \doxylink{classQCPDataSelection}{QCPData\+Selection} is used, which holds and manages multiple instances of \doxylink{classQCPDataRange}{QCPData\+Range}. In most situations, \doxylink{classQCPDataSelection}{QCPData\+Selection} is thus used.

Both \doxylink{classQCPDataRange}{QCPData\+Range} and \doxylink{classQCPDataSelection}{QCPData\+Selection} offer convenience methods to work with them, e.\+g. \doxylink{classQCPDataRange_a93529421d12fdd3a8bdb2b8061936352}{bounded}, \doxylink{classQCPDataRange_a36c8ad8acf177ffeb0a72c8d73030844}{expanded}, \doxylink{classQCPDataRange_a8a1c437f9beffd55621f48f961d7f679}{intersects}, \doxylink{classQCPDataRange_a84e1e03129dd52528efb4bac18d30183}{intersection}, \doxylink{classQCPDataRange_a279ed36602b39607699dc5652bcaf813}{adjusted}, \doxylink{classQCPDataRange_a13ca398776374a3160aa85433718b812}{contains}. Further, addition and subtraction operators (defined in \doxylink{classQCPDataSelection}{QCPData\+Selection}) can be used to join/subtract data ranges and data selections (or mixtures), to retrieve a corresponding \doxylink{classQCPDataSelection}{QCPData\+Selection}.

QCustom\+Plot\textquotesingle{}s data selection mechanism is based on \doxylink{classQCPDataSelection}{QCPData\+Selection} and \doxylink{classQCPDataRange}{QCPData\+Range}.

\begin{DoxyNote}{Note}
Do not confuse \doxylink{classQCPDataRange}{QCPData\+Range} with \doxylink{classQCPRange}{QCPRange}. A \doxylink{classQCPRange}{QCPRange} describes an interval in floating point plot coordinates, e.\+g. the current axis range. 
\end{DoxyNote}

Returns whether this range is valid. A valid range has a begin index greater or equal to 0, and an end index greater or equal to the begin index.

\begin{DoxyNote}{Note}
Invalid ranges should be avoided and are never the result of any of \doxylink{classQCustomPlot}{QCustom\+Plot}\textquotesingle{}s methods (unless they are themselves fed with invalid ranges). Do not pass invalid ranges to \doxylink{classQCustomPlot}{QCustom\+Plot}\textquotesingle{}s methods. The invalid range is not inherently prevented in \doxylink{classQCPDataRange}{QCPData\+Range}, to allow temporary invalid begin/end values while manipulating the range. An invalid range is not necessarily empty (\doxylink{classQCPDataRange_ac32e53fc05d6cd2eac96b96a7265d3b8}{is\+Empty}), since its \doxylink{classQCPDataRange_a1e7836058f755c6ab9f11996477b7150}{length} can be negative and thus non-\/zero. 
\end{DoxyNote}
