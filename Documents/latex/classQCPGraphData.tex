\doxysubsection{Detailed Description}
Holds the data of one single data point for \doxylink{classQCPGraph}{QCPGraph}. 

The stored data is\+: \begin{DoxyItemize}
\item {\itshape key\+:} coordinate on the key axis of this data point (this is the {\itshape main\+Key} and the {\itshape sort\+Key}) \item {\itshape value\+:} coordinate on the value axis of this data point (this is the {\itshape main\+Value})\end{DoxyItemize}
The container for storing multiple data points is QCPGraph\+Data\+Container. It is a typedef for \doxylink{classQCPDataContainer}{QCPData\+Container} with \doxylink{classQCPGraphData}{QCPGraph\+Data} as the Data\+Type template parameter. See the documentation there for an explanation regarding the data type\textquotesingle{}s generic methods.
