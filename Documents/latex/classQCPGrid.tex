\doxysubsection{Detailed Description}
Responsible for drawing the grid of a \doxylink{classQCPAxis}{QCPAxis}. 

This class is tightly bound to \doxylink{classQCPAxis}{QCPAxis}. Every axis owns a grid instance and uses it to draw the grid lines, sub grid lines and zero-\/line. You can interact with the grid of an axis via \doxylink{classQCPAxis_a63f1dd2df663680d2a8d06c19592dd63}{QCPAxis\+::grid}. Normally, you don\textquotesingle{}t need to create an instance of \doxylink{classQCPGrid}{QCPGrid} yourself.

The axis and grid drawing was split into two classes to allow them to be placed on different layers (both \doxylink{classQCPAxis}{QCPAxis} and \doxylink{classQCPGrid}{QCPGrid} inherit from \doxylink{classQCPLayerable}{QCPLayerable}). Thus it is possible to have the grid in the background and the axes in the foreground, and any plottables/items in between. This described situation is the default setup, see the \doxylink{classQCPLayer}{QCPLayer} documentation.

Zero lines are lines at value coordinate 0 which may be drawn with a different pen than other grid lines. To disable zero lines and just draw normal grid lines at zero, set {\itshape pen} to Qt\+::\+No\+Pen.
