\doxysubsection{Detailed Description}
\doxylink{classTCP}{TCP} Server Class. 

This \doxylink{classTCP}{TCP} Server Class is used for the data and communication interaction with Sky\+Pulse UAV ground stations. This class directly inherits QTCPServer.
